Игра "404"
Приложение для конкурса "ИграМАНиЯ" (https://vk.com/sevigramaniya).

Описание:
Игра, в которой игроку необходимо нажимать на 
идущие в порядке возрастания числа.
Цель игры - дойти до 404. 
С каждым нажатием числа меняют своё расположение, шрифт и цвет.

Разработал: 
	Кокуш Егор Константинович, g.kokush5@gmail.com.

Управление: 
        Левая кнопка мыши - нажатие по числу.
